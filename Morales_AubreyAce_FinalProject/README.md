# Study Time

## Project Overview
Study Time is a Python-based study session management application designed to help students organize and monitor their learning activities in a simple and efficient way. This program allows users to create and manage study session records by entering important details such as session ID, subject, study duration, date, and personal notes. 

The application provides a menu-driven interface that makes it easy for users to interact with the system. Users can add new study sessions, view all saved sessions, search for a specific subject, update session details, delete records, and sort sessions alphabetically. It also includes useful tracking features such as counting the total number of study sessions and calculating the total amount of time spent studying.

To improve usability, the program supports saving and loading data using JSON file handling, allowing users to keep their records even after closing the application. By combining object-oriented programming concepts, file handling, searching, sorting, and data management, Study Time serves as a practical and user-friendly tool for academic organization and productivity.

## Features
- Add Session
- View Sessions
- Search Session
- Delete Session
- Update Session
- Sort Sessions
- Count Total Sessions
- Calculate Total Study Time
- Save Data
- Load Data

2. Open terminal in the project folder.
